### **Getting Started**

1. **Clone the repository and enter the folder:**  
   `git clone https://github.com/HussainALAradi5/MedicalTrack.git && cd MedicalTrack`

2. **Install dependencies:**  
   `composer install && npm install && npm run dev`

3. **Set up the environment:**  
   `cp .env.example .env && php artisan key:generate`

4. **Edit your `.env` file** to configure database credentials and API keys as needed.

5. **Run migrations:**  
   `php artisan migrate`

6. **Start the server:**  
   `php artisan serve`

7. **Visit the app:**  
   Open your browser and go to üëâ [http://localhost:8000](http://localhost:8000)
